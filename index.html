Callbackwomen

Code women who are tech speakers.

Programming conferences, shout out with Call for Proposals when you're looking for code women to callback to it.

Tweet @callbackwomen with CFPs of professional software engineers' conferences. We also love to hear about:

  • code of conduct policy
  • proposal-writing mentorship
  • assistance for first-time speakers
  • travel funding for speakers
  • diversity scholarships
  • handicapped accessibility
  • childcare and/or nursing facilities
